# Landing Page

This directory contains the landing page for Detourist.

## Setup GitHub Pages

1. Go to your GitHub repository settings
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select "Deploy from a branch"
4. Choose "main" branch and "/docs" folder
5. Click "Save"

Your landing page will be available at: `https://YOUR_USERNAME.github.io/free-form-text-to-route/`

## Customization

Edit `index.html` to:
- Replace the video placeholder with your actual video embed code
- Update the presentation link URL
- Adjust colors, fonts, or layout as needed

## Video Embed

To embed a YouTube video:
```html
<iframe src="https://www.youtube.com/embed/YOUR_VIDEO_ID" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>
```

To embed a Vimeo video:
```html
<iframe src="https://player.vimeo.com/video/YOUR_VIDEO_ID" frameborder="0" allow="autoplay; fullscreen; picture-in-picture" allowfullscreen></iframe>
```
